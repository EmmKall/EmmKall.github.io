Tecnologías

Front-end

Tecnologías para la creación de la vista, con la cual el usuario final se comunicara con la aplicación, ofreciendo al usuario final una interfaz amigable, con el fin de que la interfaz sea facíl de usar para el usuario final

Back-end

Tecnologías para desarrollar la lógica de negocio que se requiera para que se implemente en el aplicativo, el cual se comunicara con la vista que interactua con el usuario final.

Más técnologias

Tecnologías para el almacenamiento de la información, en diferentes tipos de base de datos, sistema de versionamiento y virtualización con docker.
